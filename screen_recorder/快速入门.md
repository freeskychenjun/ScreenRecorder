# 快速入门指南

## 🚀 5分钟上手屏幕录制

### 第一步：启动程序

```bash
# 方法1：双击批处理文件
启动.bat

# 方法2：命令行运行
python main.py
```

### 第二步：选择录制方式

#### 方式A：全屏录制（最简单）

1. 直接点击 "▶ 开始录制"
2. 程序自动隐藏，开始录制全屏
3. 完成后双击托盘图标，点击 "⏹ 停止"

#### 方式B：区域录制

1. 点击 "📷 选择录制区域"
2. 在屏幕上拖动选择区域
3. 点击 "确定"
4. 点击 "▶ 开始录制"

### 第三步：调整设置（可选）

| 设置 | 推荐值 | 说明 |
|------|--------|------|
| 帧率 | 30 FPS | 适合大多数场景 |
| 编码器 | H264 | 文件小，质量好 |
| 音频 | 按需勾选 | 需要录声音时启用 |

### 第四步：查看录制结果

- 右侧表格显示录制历史
- 点击 "打开文件夹" 查看所有视频
- 右键文件可打开或删除

## 🎯 常用操作

### 暂停/继续录制

- 点击 "⏸ 暂停" 按钮暂停
- 点击 "▶ 继续" 按钮恢复
- 可多次暂停/继续

### 系统托盘操作

- 程序会自动隐藏到托盘
- **双击托盘图标**：显示窗口
- **右键托盘图标**：快捷菜单
  - 开始/停止录制
  - 打开输出文件夹
  - 退出程序

### 修改输出位置

1. 点击 "输出目录" 旁的 "浏览..." 按钮
2. 选择新文件夹
3. 确认

## ❓ 遇到问题？

### 视频没有声音

**检查清单**：
- [ ] 是否勾选了 "启用音频录制"？
- [ ] 音频设备是否被其他程序占用？
- [ ] FFmpeg 是否已安装？（运行 `ffmpeg -version` 检查）
- [ ] 麦克风是否在系统设置中启用？

### 录制卡顿

**解决方法**：
1. 降低 FPS（60 → 30）
2. 使用 MJPG 编码器
3. 关闭其他程序
4. 将输出目录设在 SSD

### 找不到录制的视频

1. 点击 "打开文件夹" 按钮
2. 查看状态栏显示的输出目录
3. 默认位置：`桌面\Recordings`

## 💡 小技巧

1. **录制前测试**：先录制10秒测试效果
2. **使用快捷键**：Ctrl+O 快速打开文件夹
3. **定期清理**：删除不需要的旧录制节省空间
4. **选择合适编码器**：H264 最节省空间
